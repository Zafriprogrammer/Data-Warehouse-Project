## Data Warehouse Components

This project implements a **Bikesales Data Mart** using a **Star Schema** design.

### Star Schema
The star schema design used in this project is provided in:
- `schema/star_schema.png`

### SQL Script
The SQL script used to create the Bikesales_DW Data Mart is located at:
- `scripts/Bikesales_DW.sql`

### Data Sources
Three cleaned Excel datasets are used as data sources for the Data Mart:
- `data/cleaned/Bikestores_2016.xlsx`
- `data/cleaned/Bikestores_2017.xlsx`
- `data/cleaned/Bikestores_2018.xlsx`

These datasets were cleaned prior to the ETL process to ensure data consistency and quality.
